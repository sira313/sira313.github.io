Mengatasi Flickering Screen Ubuntu 24.04

Thu, 01 Aug 2024

Melanjutkan postinganku sebelumnya berjudul "Lelah Menjadi Distro Hopper", ternyata cara fix flickering screen cukup dengan menambahkan sesuatu pada grub. Yup, langsung aja ke tutorialnya.

Oh iya, spesifikasi laptopku tertera pada gambar di bawah ya.

Edit file /etc/default/grub dengan text editor kesayangan kalian. Text editor yang ku gunakan adalah vim, jadi sesuaikan aja ya. Buka terminal lalu,

sudo vim /etc/default/grub

Tekan i untuk masuk ke mode edit. Cari baris seperti di bawah ini,

GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"

hapus, dan ganti dengan,

GRUB_CMDLINE_LINUX_DEFAULT="quiet splash i915.enable_dc=0 intel_idle.max_cstate=2"

Tekan esc untuk keluar dari mode edit, lalu ketik :wq lalu tekan enter. Next, reboot...

Di laptop ku sepertinya udah aman sampe sekarang gak ada flickering screen lagi. Selamat mencoba...

Terimakasih sudah mampir...
