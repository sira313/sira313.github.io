Flat palette untuk inkscape

Sun, 02 Dec 2018

Artikel ini sudah lama dan mungkin ada beberapa bagian yang sudah tidak relevan, tapi ya dicoba saja.

Flat palette ini khusus aplikasi Inkscape untuk membuat desain dengan style flat, tidak bisa dipasang di aplikasi lain. Saya membuatnya dengan melakukan picking color dari palette yang saya cari di internet (saya lupa source). Berikut adalah warna-warnanya:

Contoh pengaplikasiannya adalah pada karya yang saya bikin pada masa kuliah dulu tahun 2016 di bawah ini.

Cara install

Download dulu filenya di bawah ini

Download

Windows

Copy file ke C:\Program Files\Inkscape\share\palettes

Letak folder mungkin bisa berbeda, jadi yha cari-cari saja.

Linux Distribution

Copy to home

cp flatBySira.gpl ~/.config/inkscape/palettes/

or root directory

sudo cp flatBySira.gpl /usr/share/inkscape/palettes/

Mac OS

Gak tau, saya gak punya Macbook atau sejenis yang menjalankan Mac OS. hehe
