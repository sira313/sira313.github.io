Rapkumer adalah aplikasi yang saya kembangkan atas inisiatif pribadi untuk membantu guru dalam mengelola, menilai, dan mencetak rapor sesuai dengan Panduan Pembelajaran dan Asesmen Revisi 2025.

Aplikasi ini dirancang agar mudah digunakan, tanpa instalasi yang rumit. Cukup dijalankan di browser dan dapat diakses dari perangkat lain melalui jaringan Wi-Fi yang sama — mirip seperti cara kerja aplikasi Dapodik.

Selain itu, saya juga sudah menyesuaikan input penilaian dan format rapor ini agar sesuai dengan Panduan Pembelajaran dan Asesmen revisi 2025. Terlihat dari gambar di bawah ini, terdapat form khusus kokurikuler.

Fitur Utama

Berbasis Web Lokal

Rapkumer berjalan di browser, sehingga bisa diakses dari laptop atau HP lain yang terhubung pada jaringan Wi-Fi yang sama — tanpa perlu internet.

Impor Data Otomatis

Dapat melakukan impor data murid dan kelas dari Dapodik sehingga tidak perlu menambahkan satu per satu.
Selain itu, juga bisa impor Tujuan Pembelajaran langsung dari file Excel.

Multi Kelas dan Multi Sekolah

Satu aplikasi bisa digunakan untuk lebih dari satu kelas bahkan lebih dari satu sekolah, tanpa risiko bentrok data.

Leger

Rekapitulasi nilai akhir murid disesuaikan dengan tiga jenis kegiatan pembelajaran sesuai kurikulum merdeka yaitu intrakurikuler, kokurikuler, dan ekstrakurikuler. Bisa di download dalam format .xlsx

Cetak Rapor Format Terbaru 2025

Rapor dicetak sesuai format resmi Panduan Pembelajaran dan Asesmen Revisi 2025, termasuk kolom untuk Kegiatan Kokurikuler.

Cetak Piagam Otomatis

Tersedia fitur cetak piagam peringkat 1–4 dengan desain profesional dan siap cetak.

Antarmuka Sederhana dan Ramah Pengguna

Desain tampilan dibuat agar guru mudah memahami alur penggunaan tanpa perlu pelatihan khusus.
Klik tombol ? di navbar untuk melihat panduan penggunaan di setiap halaman.

Aplikasi ini terus dikembangkan agar semakin lengkap, stabil, dan praktis digunakan.
Tujuan utamanya sederhana:

Mempermudah guru dalam administrasi penilaian dan pelaporan hasil belajar siswa di Kurikulum Merdeka 2025.

Cara Instalasi

  1. Unduh Node.js
    Download dan instal Node.js pada komputer Anda.

  2. Unduh Aplikasi Rapkumer
    Klik tombol di bawah untuk mengunduh installer Rapkumer.

  3. Jalankan Aplikasi
    Setelah terpasang, buka Start Menu dan cari Rapkumer.
    Aplikasi akan otomatis terbuka di browser bawaan sistem.

  4. Username dan password bawaan
    Setelah aplikasi sukses terinstall maka akan meminta login. Gunakan username dan password bawaan di bawah. Ganti dan simpan di tempat aman.

Update dari versi 0.x.x

WARNING! Akan terjadi error 500 bila melakukan update dari versi 0.x.x ke versi 1.x.x karna terdapat perubahan pada database.

Jangan panik! Ikuti langkah berikut untuk melakukan migrasi database.

  1. Tekan tombol Windows + r copy paste text di bawah lalu tekan Enter.
%localappdata%/Rapkumer/run-migrations.cmd
  1. Pastikan hasil output tidak ada error seperti gambar di bawah.
  2. Jalankan ulang aplikasi Rapkumer.
