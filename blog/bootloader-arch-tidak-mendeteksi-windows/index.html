Bootloader Arch Tidak Mendeteksi Windows

Sun, 14 Jul 2024

Update 26/07/2024: Sepertinya cara ini tidak berfungsi jika kita memilih Arch Minimal saat installasi menggunakan Archinstall. Saya belum menemukan cara mengatasinya, mungkin ada package yang kurang.

Catatan ini sebagai pengingat saya. Karna setelah menginstall Arch linux, ternyata grub tidak bisa mendeteksi Windows. Oh iya, partisi /boot saya buat terpisah dari partisi EFI windows. Jadi langsung saja intinya. Saya menggunakan paru, baca dulu postingan saya yang ini. Install os-prober terlebih dulu untuk mendeteksi OS lain.

paru -S os-prober

Lalu jalankan os-prober.

sudo os-prober

Pastikan output terdapat OS yang dimaksud, dalam hal ini Windows seperti pada gambar di bawah. Langkah selanjutnya kita edit dulu file grub di /etc/defaut. Saya menggunakan vim, sesuaikan dengan teks editor kesayangan kalian.

sudo vim /etc/default/grub

Tekan i untuk masuk ke mode edit lalu gulir ke bawah ke bagian os-prober hapus tanda # seperti pada gambar di bawah. Tekan esc untuk exit dari edit mode, lalu ketik :wq lalu enter untuk menyimpan file. Terakhir, jalankan update grub.

sudo grub-mkconfig -o /boot/grub/grub.cfg

Terimakasih sudah mampir
