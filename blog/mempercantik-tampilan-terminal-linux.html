Mempercantik Tampilan Terminal Linux

Sun, 28 Jul 2024

Bosan dengan tampilan terminal gitu-gitu aja? Ini sedikit tutorial cara mempercantiknya. Mari sikat...

I use Arch btw

Jadi sesuaikan saja perintah paket managernya ya.

Ganti bash ke zsh dulu

Install zsh dulu.

paru -S zsh

Pastikan lokasi zsh.

which zsh

Misalnya punyaku ada di /usr/bin/zsh

Ganti shell default dari bash ke zsh.

chsh -s /usr/bin/zsh

Logout lalu login trus cek lagi buat pastiin.

echo $SHELL

Pastiin outputnya juga /usr/bin/zsh

Pasang tema zsh

Download dan copas fonts di bawah ini ke /usr/share/fonts

  • MesloLGS NF Regular.ttf
  • MesloLGS NF Bold.ttf
  • MesloLGS NF Italic.ttf
  • MesloLGS NF Bold Italic.ttf

Pasang tema pake powerlevel10k.

paru -S --noconfirm zsh-theme-powerlevel10k-git
echo 'source /usr/share/zsh-theme-powerlevel10k/powerlevel10k.zsh-theme' >>~/.zshrc

Tutup terminal, lalu buka terminal baru. Maka akan didapatkan tampilan seperti di bawah ini. Ikuti aja perintah dan jawab pertanyaan yang muncul. Begini contoh hasil punyaku Jika ingin mengganti style, kita bisa config ulang dengan perintah,

p10k configure

Selamat mencoba. Terimakasih sudah mampir.
