Paru - Aur Helper Wajib Setelah Install Arch

Sat, 13 Jul 2024

Arch Linux emang cool, simple. Tapi ada kurangnya, yaitu AUR (IMO) yang bakal ngeribetin penggunanya di masa depan.

Ribet gimana?

Yha nanti gimana caranya mengelola package yang terpasang dari AUR? Gimana caranya tau versi apa yang udah terinstall (maksudnya git, bin, release). Gimana dengan update massal package? Dan masalah lainnya. Di sini alasan AUR Helper ada. Ada banyak AUR Helper yang bisa dilihat di sini, tapi yang ku rekomendasikan adalah paru. Mari kita install...

  1. Install paket yang dibutuhkan dulu.
sudo pacman -S --needed base-devel git
  1. Lalu git clone paru-bin dari AUR.
git clone https://aur.archlinux.org/paru-bin.git && cd paru-bin
  1. Install ke sistem.
makepkg -si

Setelah paru terpasang, kita tidak perlu lagi menggunakan pacman bahkan untuk sekedar install, update, uninstall dan hal lain yang bisa dilakukan pacman dengan format perintah yang sama.

Terimakasih sudah mampir.
